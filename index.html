Account & Sync

Your data lives on this device first and always. The cloud is a copy, not the original — if it is unreachable, nothing stops.

Why This Key Is Safe

The old JsonBin key was a bearer token: whoever read it owned the data outright, and it was sitting in a public repository. The Supabase anon key is public by design — on its own it can read nothing, because Row Level Security refuses every row that is not yours. The key gets you to the door. Your login opens it.

That guarantee is worth exactly nothing if RLS is off. Run the setup SQL before trusting a byte to this.

XP Audit

Every XP award writes a receipt. This checks the balance against the receipts. If they disagree, something wrote XP without logging it — and that is a bug, not a bonus.

Reset Character

Wipes XP, level, wallet and ledger. Keeps every objective, task and stage. The work was real; only the number describing it was wrong. Levels were never the point — you can climb again in a fortnight.

Export

A full, readable dump of everything the Forge knows. Hand it to an analyst — human or otherwise — and ask what it says about you. The numbers do not flatter and they do not lie.
The report is written for analysis: it carries the schedule, the session history with every multiplier that fired, the rock record, the wallet ledger and the reward economy. The raw JSON is the complete state, for a full restore or a deeper dig.
